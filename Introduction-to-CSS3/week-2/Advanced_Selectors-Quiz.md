# Advanced Selectors Quiz

### Question 1
Assume that this code is linked together correctly. 

    p{
      color: red;
      padding:10px 5px;
      background: black;
    }
    .fancy{
      font-family: cursive;
      background: red;
      color: green;
    }
    .plain{
      font-family: Times, serif;
      color: black;
    }  
<br>

    <p>Hi</p>
    
 What color font is used to display "Hi"?
 
    1) red
    2) green
    3) black
    4) browser default
   
_ ANS: 3) black_<hr>

### Question 4
